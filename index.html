Structure

Where do I put my slides HTML code?

Just put it inside the element with the calss .main-container

How do I create a Slide

Every Slide must be a child of the .main-container. Each slide is at least 100% tall, but can get taller (then you can just simply scroll down).

There are different kind of slides. To create a simple content slide with padding (like this one), use the class .content-slide like this:

Alternatively you can style them completely yourself.

<div class="slide content-slide">
  <h2>Title</h2>
  <p>Lorem ipsum...</p>
</div>

Controls

Previous slide

  • Pressing
  • Scrolling upwards beyond the slide start
  • pressing the "up" button (orange, top right)

Next slide

  • Pressing
  • Scrolling down beyond the slide end
  • pressing the "down" button (orange, bottom right)

Number keys

You can also press the numbers 1 to 9 with one being the first, nine being the ninth (and zero being the tenth slide).

Additional info:
If you have n slides each number key k > n will scroll to page n instead of k. For example if you have only 4 pages, pressing 5 will bring you to page 4 (because 5 doesn't exist).

Scrolling in long slides or small displays

If the slide is larger than the display area, you either have to trim some fat or bite the bullet and scroll down while presentating... But be careful! if you scroll past the bottom of the slide you will be taken to the next slide.

"Settings"

In the JS file you will find a settings section. You can easily set the following parameters:

Name Description Default
keysForDown keys to press to scroll to next slide (seperated by spaces) "pagedown enter return down right space"
keysForUp keys to press to scroll to previous slide (seperated by spaces) "pageup up left backspace"
scrollMargin pixels you can scroll "beyond" a slide before you trigger the animation to the next one 20
wrap when the last slide is reached, warp around to the first one false
animationTime page scroll animation time 500
hljsLangs highlight.js languages (seperated by ", ") "javascript, html, css"
hljsTabReplace highlight.js tab replace "  " (two spaces)
