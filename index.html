Installing environment in VSCode

Download VSCode

Visual Studio Code (VSCode) is a source code editor developed by Microsoft for Windows, Linux, macOS. It is free to download. You can download it from its official website. Download Link


Setup extension in VSCode

After successful installation click on widgets this icon from the left bar or simply press Ctrl+Shift+X to go to extensions. In search bar type C# and an extension will appear. Install that extension. This extension will provide the following features inside VSCode -

  • Lightweight development tools for .NET Core
  • Great C# editing support including Syntax Highlighting, IntelliSense, Go to Definition, Find All References etc.
  • Debugging support for .NET Core (CoreCLR)
  • Support for project.json and csproj projects on Windows, Linux and macOS

You can also install another extension named as C# Extension. It will provide extensions to the IDE that will hopefully speed up your development workflow. Now you are good to go.

Project structure

Create a Project

Open the terminal from the folder or directory in which you want to create the project. Lets say you want to create a project named musicPlayer. The following commnad will create a project for you. dotnet new mvc -o musicPlayer Then you need to change your directory to that project and open it in VSCode. After opening the project you will see a notification on the right bottom corner. Click on Yes button.

Next slide

  • Pressing
  • Scrolling down beyond the slide end
  • pressing the "down" button (orange, bottom right)

Number keys

You can also press the numbers 1 to 9 with one being the first, nine being the ninth (and zero being the tenth slide).

Additional info:
If you have n slides each number key k > n will scroll to page n instead of k. For example if you have only 4 pages, pressing 5 will bring you to page 4 (because 5 doesn't exist).

Scrolling in long slides or small displays

If the slide is larger than the display area, you either have to trim some fat or bite the bullet and scroll down while presentating... But be careful! if you scroll past the bottom of the slide you will be taken to the next slide.

"Settings"

In the JS file you will find a settings section. You can easily set the following parameters:

Name Description Default
keysForDown keys to press to scroll to next slide (seperated by spaces) "pagedown enter return down right space"
keysForUp keys to press to scroll to previous slide (seperated by spaces) "pageup up left backspace"
scrollMargin pixels you can scroll "beyond" a slide before you trigger the animation to the next one 20
wrap when the last slide is reached, warp around to the first one false
animationTime page scroll animation time 500
hljsLangs highlight.js languages (seperated by ", ") "javascript, html, css"
hljsTabReplace highlight.js tab replace "  " (two spaces)
